#								Yuhasz's Calculator											#
#############################################################################################

Version v0.1

This Will Do The Following:

1.) Pull JSON data from RSBuddy
2.) Create a database in sqllite3 from JSON data provided
3.) Use the database for further querying. (Theres a couple of different ways I queried the db)
4.) Python is used extensively in "calc_module.py" to breakdown Queryset data into lists.
5.) Django is used to present the data in HTML format.
6.) The end product will show the profits made from doing said crafting in OSRS

##############################################################################################
#								  Working tablesets											 #
